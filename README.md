## Getting Started

### Installation

This project is using [nvm](https://github.com/nvm-sh/nvm) to manage Node.js version.

```bash
nvm use
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
